# Harry Potter App 🪄
# Created by: IMAN ✨

یک اپلیکیشن جادویی برای عاشقان هری پاتر با تم تاریک و کلاسیک

## ویژگی‌های اصلی ✨

- 🎨 **تم تاریک و مینیمال** - طراحی کلاسیک و جذاب
- 🔤 **فونت‌های فارسی** - Iran Sans و Vazir
- ✨ **انیمیشن‌های سرگرم کننده** - اثرات بصری جذاب
- 👥 **معرفی شخصیت‌ها** - اطلاعات کامل درباره شخصیت‌ها
- 🎮 **بازی هورکراکس‌ها** - بازی تعاملی و جذاب
- 📱 **رابط کاربری مدرن** - طراحی responsive و قشنگ

**Creator: IMAN** 💎

## ساختار پروژه 📁

```
lib/
├── main.dart                 # نقطه ورود اپلیکیشن
├── theme/
│   └── app_theme.dart       # تم برنامه
├── models/
│   ├── character.dart       # مدل شخصیت
│   └── horcrux.dart        # مدل هورکراکس
├── providers/
│   └── app_provider.dart   # State management
└── screens/
    ├── home_screen.dart           # صفحه اول
    ├── characters_screen.dart     # لیست شخصیت‌ها
    ├── character_detail_screen.dart # جزئیات شخصیت
    └── horcrux_game_screen.dart   # بازی هورکراکس‌ها
```

## تکنولوژی 🛠️

- **Flutter** - چارچوب توسعه
- **Provider** - State Management
- **Go Router** - Navigation
- **Flutter Animate** - انیمیشن‌ها
- **Hive** - Local Storage

## شخصیت‌های موجود 👥

1. **هری پاتر** - پسری که زندگی کرد
2. **هرمیون گرنجر** - باهوش‌ترین جادوگر
3. **رن ویزلی** - بهترین دوست
4. **آلبوس دامبلدور** - جادوگر حکیم
5. **سیویروس اسنیپ** - استاد معمولات
6. **دراکو مالفوی** - رقیب هری
7. **لونا لاوگود** - رویاپرداز رَیونکلو
